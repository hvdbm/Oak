# Oak

Tree generated with [Graphviz](https://graphviz.org).

## Configuration file

A configuration `JSON` file can be used to customize the tree. The file is optionnal and by default, the values of the tree will :

```json
{
  "filename": "family_tree.png",
  "graph_title": "",
  "node_shape": "box",
  "show_nickname": false
}
```
